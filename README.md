# 🗂️ 20 Proje Koleksiyonu

## 📌 Genel Bakış  
Bu repo, **20 farklı mini proje** içermektedir. Projelerde, farklı teknolojileri (ADO.NET, Entity Framework, Dapper, Web API, RapidAPI, MongoDB, PostgreSQL, JWT, Kaggle Dataset, Ollama AI vb.) kullanarak geliştirdim.  

Amacım, **farklı veri tabanları, ORM araçları, API tüketimi ve modern .NET teknolojilerini** gerçek dünya senaryoları ile geliştirmek.  

---

## 🛠️ Kullanılan Teknolojiler  
- **Backend / ORM:** ADO.NET, Entity Framework (DbFirst & Code First), Dapper ORM  
- **API / Web:** ASP.NET Core Web API, Swagger, RapidAPI  
- **Veritabanları:** SQL Server, PostgreSQL, MongoDB  
- **Kimlik Doğrulama:** JWT (Json Web Token)  
- **Veri Analizi:** Kaggle Datasets, Chart Controls  
- **AI:** Ollama AI (phi3:mini modeli)  
- **Diğer:** MailKit, MimeKit, Kaggle
